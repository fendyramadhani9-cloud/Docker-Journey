# Docker Documentation

Folder ini berisi seluruh materi pembelajaran Docker yang disusun berdasarkan tingkat kesulitan (Phase).

Setiap Phase memiliki kumpulan module yang saling berhubungan, sehingga materi dapat dipelajari secara bertahap mulai dari dasar hingga tingkat lanjut.

---

## 📚 Phase 01 — Docker Fundamentals

Phase ini membahas dasar-dasar Docker yang wajib dipahami sebelum masuk ke materi yang lebih kompleks.

Materi yang dipelajari meliputi:

- Docker Engine
- Container Lifecycle
- Docker Networking
- Port Mapping
- Dockerfile
- Image Build
- Storage (Volume & Bind Mount)
- Docker Compose
- Container Management

📁 Folder:
```
phase-01-fundamentals/
```

---

## 🚀 Phase 02 — Docker Intermediate

Phase ini berfokus pada penggunaan Docker dalam pengembangan aplikasi yang lebih nyata.

Materi akan membahas topik seperti:

- Environment Variables
- Docker Hub
- Docker Registry
- Multi Container Application
- Dockerfile Best Practices
- Image Optimization
- Docker Networking (Advanced)

📁 Folder:
```
phase-02-intermediate/
```

---

## ⚙️ Phase 03 — Docker Advanced

Phase ini membahas praktik Docker yang umum digunakan pada lingkungan production.

Materi akan mencakup:

- Docker Security
- Troubleshooting
- Production Tips
- Deployment Strategy
- Final Project

📁 Folder:
```
phase-03-advanced/
```
