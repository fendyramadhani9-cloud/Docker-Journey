# 🔍 Verifikasi Instalasi

Pastikan Docker dapat dijalankan menggunakan perintah berikut.

```bash
docker --version
```

```bash
docker compose version
```

```bash
docker ps
```

Apabila seluruh command berhasil dijalankan tanpa error, berarti environment Anda siap digunakan.

# 📂 Workspace

Disarankan membuat satu folder khusus untuk mengerjakan seluruh Hands-on Labs.

Contoh:

```text

```

Seluruh file latihan, Dockerfile, maupun project nantinya dapat dikerjakan di dalam folder tersebut agar lebih mudah dikelola.

# 🤖 Journey CLI

Saat ini Journey CLI masih dalam tahap pengembangan.

Pada versi berikutnya, seluruh Lab dapat divalidasi menggunakan command berikut.

```bash
journey check docker phase-01 lab-01
```

Journey CLI akan:

- Memvalidasi hasil pekerjaan.
- Memberikan skor.
- Menampilkan requirement yang belum terpenuhi.
- Memberikan rekomendasi materi apabila belum lulus.
