# 📋 Misi

Selesaikan seluruh tugas berikut.

1. Pastikan Docker Image **ubuntu** tersedia pada komputer Anda.
2. Buat sebuah Container dengan nama **ubuntu-lab**.
3. Jalankan Container tersebut.
4. Pastikan Container berhasil berjalan.
5. Verifikasi bahwa Container dapat dijalankan kembali apabila sebelumnya berhenti.

> **Catatan**
>
> Anda bebas menggunakan command apa pun selama seluruh misi berhasil diselesaikan.
